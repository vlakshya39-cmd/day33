# 🚀 Day 33 - Media Integrity Analyzer

## 📌 Project Overview

Today I built **Media Integrity Analyzer**, an interactive educational web application that teaches users how to identify misleading headlines, emotional manipulation, and media bias through realistic scenarios.

The application provides a hands-on learning experience where users analyze news headlines and social media posts before receiving expert feedback and media literacy insights.

---

## ✨ Features

- 📰 Headline Detective Challenge
- 🎭 Emotion Detection Challenge
- 📊 Live Media Integrity Metrics
- 📈 Overall Media Integrity Dashboard
- 🎯 Randomized Practice Scenarios
- 🌙 Modern Responsive Interface
- 🔄 Replay Functionality
- 📱 Mobile Friendly Design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
